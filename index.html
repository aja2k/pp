Projects

A사 차세대 IT시스템 구축 POC

차세대 IT 시스템 구축을 위한 POC 프로젝트에서 레거시 시스템의 클라우드 네이티브 전환. Linux와 Kubernetes 기반의 하이브리드 클라우드 환경을 설계하고, 기존 모놀리식 .NET 애플리케이션을 컨테이너 기반 Spring Boot 서비스로 현대화. GitOps 기반의 CI/CD 파이프라인을 구축하여 멀티 클러스터 환경(총 10개)에서의 자동화된 배포를 실현했으며, Terraform, Ansible, Helm 등 IaC 도구를 활용하여 인프라 프로비저닝을 자동화.
